## Additional Information

Module 4 activity focuses on how user events are handled through PySide6's
even-driven programming.

The following components were applied in completing activity 4.

Event Handling - Handlers or listeners that responds user interactions such as clicks and button selections.

Slots - Method that responds to signals, such as adding or removing contacts
when buttons are clicked.

Signals - Connects events from widgets such as "clicked" from "QPushButton"
to trigger methods and actions.

PySide6 - core Graphica User Interface (GUI) elements to create a functional
contact management interface. i.e: QLineEdit, QPushButton, QMessageBox, and more.
